## ğŸ“– Overview
**VoiceAccess** is a specialized web-based audio library designed specifically for visually impaired and color-blind university students. Traditional digital libraries rely on keyboards and mice, which are significant barriers for blind users. 

VoiceAccess eliminates these hardware dependencies by introducing a **"Contactless Interaction Model"** using **Face Recognition** for security and **Hand Gestures** for navigation.
